1. Название проекта
● SecureMayatnik
2. Проблема или Возможность
● Пользователи часто сталкиваются со сложностью создания надежных, уникальных и запоминающихся логинов и паролей для различных сервисов. Использование простых или повторяющихся комбинаций создает угрозу безопасности, а запоминание сложных паролей — неудобство. Данный продукт решает эту проблему, предоставляя инструмент для быстрой и безопасной генерации учетных данных с гибкими настройками.
3. Цели проекта
●Предоставить пользователю интуитивно понятный веб-интерфейс для генерации логинов и паролей.

●Реализовать гибкие настройки генерации (длина, использование символов, исключение похожих символов).

●Обеспечить базовую оценку надежности сгенерированных паролей.

●Создать функционал для удобного копирования и кратковременного хранения сгенерированных данных на клиенте.
4. Основной функционал
●Генерация пароля: Создание случайного пароля с настройкой длины (8-32 символа) и типов используемых символов (прописные/строчные буквы, цифры, специальные символы).

●Генерация логина: Создание случайного логина на основе выбранного шаблона (например, "имя_существительное + число", "прилагательное_существительное" и т.д.).

●Комбинированная генерация: Одновременная генерация связанной пары "логин + пароль".

●Быстрое копирование: Кнопка "Скопировать в буфер обмена" для каждого сгенерированного значения.

●Интерфейс управления: Простой и чистый веб-интерфейс с ползунками, чекбоксами для настроек и кнопкой генерации.
5. Целевая аудитория
●Обычные пользователи интернета, регистрирующиеся на новых сайтах.

●Системные администраторы и разработчики, создающие тестовые аккаунты.

●Специалисты по информационной безопасности, нуждающиеся в быстром создании надежных учетных данных для демонстраций или тестов.
6. Технологический стек
● Frontend (Клиентская часть): HTML5, CSS3, JavaScript (ES6+). Выбор обоснован универсальностью, отсутствием необходимости в компиляции и возможностью запуска в любом браузере.
7. Основные модули системы
●UI Module (Модуль интерфейса): Отвечает за отрисовку всей веб-страницы, кнопок, полей ввода и отображения результатов. Обрабатывает действия пользователя.

●Generator Core (Ядро генератора): Содержит логику генерации случайных последовательностей. Отдельные подмодули для паролей и логинов.
8. Критерии успеха
● Реализован весь заявленный функционал
● Интерфейс интуитивно понятен, пользователь может сгенерировать данные за 3 клика.
